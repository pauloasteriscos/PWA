## Protótipo:
3-Prototipo

Segundo a recomendação, foi utilizado o HTML5, que permite uma construção evolutiva, caso avance com o projecto para uma estrutura com algum framework como o vue.js, por exemplo.

Inclui protótipos navegáveis em HTML5 para ambas as novas entidades:
- Listagem
- Criação
- Edição
- Visualização

Foi utilizado um ficheiro CSS comum, "animalec-backoffice.css", para manter consistência visual entre todas as páginas.

O protótipo é funcional do ponto de vista visual, mas sem back-end, somente a exemplo de protótipo.
